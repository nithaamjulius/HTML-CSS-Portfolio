## Challenges

A big challenge I faced during this project was ironically being to ambitious with my styling ideas. An example of this would be that my original idea was to purely through css style a book so that it could flip through multiple pages showing off my different projects. I'm sure it can be done but given the amount of time given for the project and my current HTML + CSS skills I was not confident that I would be able to finish this and I in hindsight am happy I made this decision. 
## Lessons Learned & Future Improvements

Throughout this the process of making this project is that I still struggle with having a clean concise way of structuring/styling things when it comes to a general website. My biggest aid in this though was looking for inspiration on other websites through platforms/apps like Canva and Pinterest. I also need to deepen my CSS knowledge and make sure that I don't miss any typos or formatting issues within HTML.
